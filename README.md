# [ToDo](https://todo-eosin-three.vercel.app/)

Simple To do list

## Features

- Mobile layout
- Sort tasks

## Tech Stack

- React
- Next
- Typescript
- Eslint
- Heroicons
- Vercel
- Tailwind
- Jest

## How to use

1. Clone the repo

```bash
git clone https://github.com/SavitskayaKseniya22/todo.git
```

2. Change directory

```bash
cd todo
```

2. Install dependencies

```bash
npm install
```

3. Start the dev server

```bash
npm run dev
```
